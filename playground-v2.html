💡 Quick Tips

  • 1) Click a device on the left, then click canvas to place
  • 2) Or drag a device from the left directly onto canvas
  • 3) Press L and click two nodes to create a link
  • 4) In Select mode, drag nodes to reposition them
  • 5) Click any node or link to edit properties on the right
  • 6) Press Simulate to run this topology
  • V Select mode
  • N Add node mode
  • Del Delete selected
  • Ctrl+Z Undo
  • Ctrl+Y Redo
