# Icon placeholders

Place your downloaded SVG/PNG icons in this `icons/` folder using these filenames so the canvas can render them:

- `router.svg` – core/backbone routers
- `switch.svg` – access/switch nodes and classrooms
- `pc.svg` – workstation endpoints
- `lab.svg` – XR/VR labs or maker spaces
- `sensor.svg` – IoT / sensor clusters
- `server.svg` – compute nodes or local render boxes
- `firewall.svg` – security/edge firewalls
- `wifi.svg` – wireless controllers or guest gateways
- `cloud.svg` – external cloud services
- `globe.svg` – internet uplink visualization

Any custom node you create can reference these names (or add more filenames here and point to them in the “Icon name” field). If an icon is missing, the app will fall back to a colored dot.
